# Bright Star Academy Website

**Singapore's First Drama-STEM Enrichment Programme**

🌐 Live at: [brightstaracademy.com](https://brightstaracademy.com)

## About

Bright Star Academy offers a groundbreaking approach to early childhood education that blends drama, storytelling, and STEM learning for children ages 3-6.

- **Little Tinkerers** (Ages 3-4): Play-based learning through sensory exploration
- **Star Explorers** (Ages 5-6): Inquiry-based learning with scientific experiments

## Tech Stack

- Static HTML/CSS/JS

- Google Fonts (M PLUS 1, Poppins)
