# Brightstar Academy Website

Modern, fast website for Brightstar Academy — Drama-STEM enrichment for children ages 3-6.

## Tech Stack
- Pure HTML/CSS/JS (no frameworks needed)
- Hosted on Vercel (free tier)
- Calendly integration for bookings

## Pages
- `index.html` — Homepage
- `free-lessons.html` — Free trial booking page
- `styles.css` — Global styles

## Development
Simply open `index.html` in a browser to preview locally.

## Deployment
Pushes to `main` branch auto-deploy via Vercel.
